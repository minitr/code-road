# 207. 课程表 (opens new window)

第一种做法，按照拓扑排序的经典套路，直接完成

  1. 进行一次遍历，计算所有课程的入度，其中如果prerequisites[i][0]和prerequisites[i][1]相同，那就直接返回false，因为已经构成环了
  2. 对计算好的入度遍历，取出入度为0的课程放入队列作为起点
  3. 取出队列中的课程，将其对应的边的终点入度减一，如果该终点入度为0了，那么就放进队列作为起点
  4. 遍历入度数组，如果有不为0的直接返回false
public boolean canFinish(int numCourses, int[][] prerequisites) {
        int[] classes = new int[numCourses];
        for (int i = 0; i < prerequisites.length; i++) {
            if (prerequisites[i][0] == prerequisites[i][1]) {
                return false;
            }
            int classFront = prerequisites[i][1];
            classes[classFront]++;
        }
        Deque<Integer> classDeque = new ArrayDeque<>();
        for (int i = 0; i < numCourses; i++) {
            if (classes[i] == 0) {
                classDeque.addLast(i);
            }
        }
        System.out.println(classDeque);
        while (!classDeque.isEmpty()) {
            int classFront = classDeque.pollLast();
            for (int i = 0; i < prerequisites.length; i++) {
                if (classFront == prerequisites[i][0]) {
                    classes[prerequisites[i][1]]--;
                    if (classes[prerequisites[i][1]] == 0) {
                        classDeque.addLast(prerequisites[i][1]);
                    }
                }
            }
        }
        for (int i = 0; i < numCourses; i++) {
            if (classes[i] != 0) {
                return false;
            }
        }
        return true;
    }

这里还有可以优化的复杂度，应该在求入度的同时给起点设置对应的终点list，便于队列弹出起点后的进行的入度减少步骤的遍历，所以修改如下

public boolean canFinish(int numCourses, int[][] prerequisites) {

            classList.add(new ArrayList<>());
        }
        for (int i = 0; i < prerequisites.length; i++) {
            if (prerequisites[i][0] == prerequisites[i][1]) {
                return false;
            }
            classList.get(prerequisites[i][0]).add(prerequisites[i][1]);
            int classFront = prerequisites[i][1];
            classes[classFront]++;
        }
        System.out.println(classList);
        Deque<Integer> classDeque = new ArrayDeque<>();
        for (int i = 0; i < numCourses; i++) {
            if (classes[i] == 0) {
                classDeque.addLast(i);
            }
        }
        while (!classDeque.isEmpty()) {
            int classFront = classDeque.pollLast();
            List<Integer> list = classList.get(classFront);
            for (int i = 0; i < list.size(); i++) {
                int back = list.get(i);
                classes[back]--;
                if (classes[back] == 0) {
                    classDeque.addLast(back);
                }
            }
        }
        for (int i = 0; i < numCourses; i++) {
            if (classes[i] != 0) {
                return false;
            }
        }
        return true;
    }

# 459. 重复的子字符串 (opens new window)

给定一个非空的字符串 s ，检查是否可以通过由它的一个子串重复多次构成。

既然是要判断重复多次构成，那么子串长度必是总长的因子，只需要遍历s.size()的因子，判断是否满足条件即可

     bool repeatedSubstringPattern(string s) {
        int n=s.size();
        for(int i=1;i<n;++i){
            if(n%i==0){
                if(isValid(s,i)){
                    return true;
                }
            }
        }
        return false;
    }
    bool isValid(string s,int index){
        int k=s.size()/index;
        for(int i=0;i<k;i++){
            for(int j=0;j<index;j++){
                if(s[i*index+j]!=s[j]){
                    return false;
                }
            }
        }
        return true;
    }

# 1790. 仅执行一次字符串交换能否使两个字符串相等 (opens new window)

一次遍历统计字符串中不相同的位置，并计数，如果数目不为2直接返回false，如果为0直接返回true，接下来再比较这两个位置的字符即可

    bool areAlmostEqual(string s1, string s2) {
        int index1=-1;
        int index2=-1;
        int count=0;
        for(int i=0;i<s1.size();++i){
            if(s1[i]!=s2[i]){
                ++count;
                if(count>2){
                    return false;
                }
                if(index1==-1){
                    index1=i;
                }else{
                    index2=i;
                }
            }
        }
        if(count==0){
            return true;
        }
        if(count==1){
            return false;
        }
        return s1[index1]==s2[index2]&&s2[index1]==s1[index2];
    }

# 1037. 有效的回旋镖 (opens new window)

给定一个数组 points ，其中 points[i] = [xi, yi] 表示 X-Y 平面上的一个点，如果这些点构成一个 回旋镖 则返回 true

调用一下自定义排序试试

class Solution {
public:
    bool isBoomerang(vector<vector<int>>& points) {
        sort(points.begin(),points.end(),cmp);
        int a=points[1][1]-points[0][1];
        int b=points[1][0]-points[0][0];
        int c=points[2][1]-points[1][1];
        int d=points[2][0]-points[1][0];
        return !(a*d==c*b);
    }
    static bool cmp(const vector<int>& point1,const vector<int>& point2){
        if(point1[0]==point2[0]){
            return point1[1]<point2[1];
        }
        return point1[0]<point2[0];
    }
};

在过程中出现了这个错误，是因为sort的第三个参数，也就是我们自定义的cmp，只允许接收两个参数，而当在类内定义cmp函数时，如果像上述那样定义，在编译时，编译器会向函数添加隐式参数this，而变成了三个参数。

解决方法：

  1. 定义在类外
  2. 添加static关键字
  3. 使用lambda表达式声明

# 1232. 缀点成线 (opens new window)

判断一系列点是否在坐标系中属于同一条直线上。

最后一道用Java写了，也是一样的排序，再全部与起点进行斜率计算，为了避免出现小数，斜率的比较直接采用乘法

   public boolean checkStraightLine(int[][] coordinates) {
        if (coordinates.length <= 2) {
            return true;
        }
        Arrays.sort(coordinates, (o1, o2) -> {
            if (o1[0] == o2[0]) {
                return Integer.compare(o1[1], o2[1]);
            }
            return Integer.compare(o1[0], o2[0]);
        });
        int a = coordinates[1][1] - coordinates[0][1];
        int b = coordinates[1][0] - coordinates[0][0];
        for (int i = 2; i < coordinates.length; ++i) {
            int c = coordinates[i][1] - coordinates[0][1];
            int d = coordinates[i][0] - coordinates[0][0];
            if (a * d != c * b) {
                return false;
            }
        }
        return true;
    }

# 2078. 两栋颜色不同且距离最远的房子 (opens new window)

直接暴力遍历，顺便进行一个小小的去重，数据量大的话可以考虑使用set

    int maxDistance(vector<int>& colors) {
        int length=0;
        int j=0;
        int n=colors.size();
        int ans=0;
        for(int i=0;i<n;++i){
            if(i>0&&colors[i]==colors[i-1]){
                continue;
            }
            length=0;
            j=i+1;
            while(j<n){
                if(colors[j]!=colors[i]){
                    length=max(length,j-i);
                }
                ++j;
            }
            ans=max(ans,length);
        }
        return ans;
    }

# 1323. 6 和 9 组成的最大数字 (opens new window)

获取每一位数字并从高位开始向后遍历，遇到6变9并修改flag值即可

    int maximum69Number (int num) {
        vector<int> res;
        while(num){
            res.push_back(num%10);
            num/=10;
        }
        bool flag=false;
        for(int i=res.size()-1;i>=0;--i){
            if(res[i]==6&&!flag){
                res[i]=9;
                flag=true;
            }
            num=num*10+res[i];
        }
        return num;
    }

# 942. 增减字符串匹配 (opens new window)

这题甚至卡了一会，其实思路很简单，要把I和D分开来想，单独来看I，就是一个递增的数组，单独来看D，就是一个递减的数组，且D的位置一定比I大，之后无非就是遍历，设置一个值为n一个值为0，分别往D和I添加即可

    vector<int> diStringMatch(string s) {
        int n=s.size();
        vector<int>ans;
        int k=0;
        int j=n;
        for(int i=0;i<n;++i){
            if(s[i]=='I'){
                ans.push_back(k);
                ++k;
            }else{
                ans.push_back(j);
                --j;
            }
        }
        if(s[n-1]=='D'){
            ans.push_back(j);
        }else{
            ans.push_back(k);
        }
        return ans;
    }

贪心之处就在于，每次都是选剩余数字中的最小数和最大数

# 541. 反转字符串 II (opens new window)

考虑清楚边界条件，反转的时候采用双指针进行反转

最后进行判断：当剩余字符少于k个，也就是n-1-i+1<k即i>n-k时，全部反转

    string reverseStr(string s, int k) {
        int i=0;
        int n=s.size();
        while(i+2*k<=n){
            reverse(s,i,i+k-1,k);
            i+=2*k;
        }
        if(i>n-k){
            reverse(s,i,n-1,k);
        }else{
            reverse(s,i,i+k-1,k);
        }
        return s;
    }
    void reverse(string &s,int i,int j,int k){
        while(i<=j){
            char c=s[i];
            s[i]=s[j];
            s[j]=c;
            ++i;
            --j;
        }
    }

# 面试题 16.24. 数对和 (opens new window)

两数之和精简版，往中间遍历即可

    vector<vector<int>> pairSums(vector<int>& nums, int target) {
        int n=nums.size();
        vector<vector<int>> ans;
        if(n==0||n==1){
            return ans;
        }
        sort(nums.begin(),nums.end());
        int i=0,j=n-1;
        while(i<j){
            int sum=nums[i]+nums[j];
            if(sum==target){
                ans.push_back({nums[i],nums[j]});
                ++i;
                --j;
            }else if(sum<target){
                ++i;
            }else{
                --j;
            }
        }
        return ans;
    }

# 696. 计数二进制子串 (opens new window)

我觉得一个简单题的tag配不上它....

  1. 首先利用动态dp数组求得每个位置的最长相同子串，以00110011为例，求出来的dp即为12121212
  2. 遍历dp数组，以0011中的第一个1为例，对应的dp值为1，那么就取对应的i-dp位置的字符，发现与当前i位置不同，这就说明有可能存在满足条件的子串，需要再进一步判断dp[i-dp[i]]与dp[i]的大小，如果前者大于等于后者，说明满足情况，可以+1

代码：

    int countBinarySubstrings(string s) {
        if(s.size()==1){
            return 0;
        }
        int n=s.size();
        int dp[n];
        memset(dp,0,n);
        dp[0]=1;
        for(int i=0;i<n;++i){
            if(i>0&&s[i]==s[i-1]){
                dp[i]=dp[i-1]+1;
            }else{
                dp[i]=1;
            }
        }
        int ans=0;
        for(int i=0;i<n;++i){
            int index=i-dp[i];
            if(index>=0&&s[index]!=s[i]){
                if(dp[index]>=dp[i]){
                    ++ans;
                }
            }
        }
        return ans;
    }
